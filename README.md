send//it 🪨
A personal climbing training log. Log sessions, track exercises, and visualise training focus and progression over time.
Live: neysetas.github.io/SendIt

Features

Session logging — pick exercises from a categorised sidebar, fill in weight, sets, reps, time, or duration
Cross-device sync — data stored in Supabase, accessible from any device after login
Training focus analytics — see what percentage of your load has been finger, strength, power, endurance, technique, or mobility work, across any time window
Exercise evolution graphs — track weight or time progression over time for trackable exercises (hangboard, pull-ups, lock-offs, etc.)
Session history — full log of past sessions with notes and category breakdown

Exercise categories

🟠 Fingers 
🟢 Strength 
🔴 Power 
🟣 Technique 
🔵 Endurance 
🟡 Mobility 

Tech stack

HTML + CSS + JavaScript — no framework, no build step
Supabase — auth and database
Chart.js — graphs
Hosted on GitHub Pages

Local development
Just open index.html in a browser. No build tools needed.

Deployment
Push to the main branch — GitHub Pages serves it automatically.
git add index.html
git commit -m "your message"
git push
