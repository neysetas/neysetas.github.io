How training load is calculated

Every exercise you log contributes points to its categories. The percentages in the charts show each category's share of total points.

Set-based categories (Fingers, Strength): each set is worth a configurable number of points. Default is 1 pt/set. Finger sets are more demanding, so you might give them 1.5 pts.

Time-based categories (Power, Technique, Endurance, Mobility): you set how many minutes equal 1 point. Default is 5 min = 1 pt, so 30 min of ARC = 6 pts — comparable to 6 pull-up sets.

Category multipliers per exercise: for exercises with multiple categories, instead of splitting points equally you can assign a relative weight to each category. A multiplier of 1.0 is the baseline. So if Moonboard has finger=1.5 and technique=0.5, fingers get 3× more load than technique from that exercise. The total points are still based on the time logged — the multipliers just control how those points are distributed.

Restore defaults resets all values to the recommended balanced starting point.
